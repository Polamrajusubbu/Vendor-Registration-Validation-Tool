📋 Vendor Registration Form Automation using Microsoft Word & VBA

🚀 Overview

This project is a Microsoft Word-based vendor registration system designed to capture and validate vendor details through a structured document. It enhances data accuracy, ensures completeness, and standardizes the vendor onboarding process.

---

🧩 Problem Statement

Vendor details were collected through unstructured documents or emails, leading to:

- Inconsistent formats
- Missing information
- Manual validation effort
- Data entry errors

---

💡 Solution

Developed a Word-based automated form that:

- Captures vendor details in a structured format
- Guides users with predefined input fields
- Ensures required information is provided
- Standardizes vendor data collection

---

✨ Key Features

- 📝 Structured vendor input form in Word
- 📌 Predefined fields for consistent data entry
- ⚠️ Validation logic for required inputs
- 🎯 User-friendly document layout
- 📂 Document-based submission format
- 🔒 Controlled editing behavior

---

⚙️ Technical Implementation

🔹 Microsoft Word

- Designed structured form layout
- Used content controls / formatted sections
- Guided user input with placeholders

---

🔹 VBA Automation

- Input validation logic
- Field checks for mandatory data
- Controlled document behavior
- Automation of form handling

---

🔄 Workflow

User fills Word Form → Validation → Submission → Structured Data Capture

▶️ How to Use

Vendor-

1. Open the Word template
2. Fill in vendor details
3. Complete required fields
4. Submit the document

---

User-

1. Open the Word template
2. Press ALT + E to extract vendor data
3. Select the path for Vendor Master excel file
5. Check the Vendor Master and use for further ERP update
